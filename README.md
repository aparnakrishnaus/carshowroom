# Car Showroom Management - Django Web Application

A comprehensive Django-based web application to manage car showroom operations, including cars, spare parts, bookings, orders, payments, and admin management.

## Features
### User Side
- **User Authentication:** Register, Login, Logout, Password Reset & Forgot Password
- **Profile Management:** View and manage bookings, update profile
- **Car Booking:** Book cars with advance payment option
- **Test Drive Booking:** Schedule test drives
- **Spare Part Booking:** Book spare parts
- **Service Booking:** Book services
- **Payments:** Razorpay integration for online payments
- **Search Functionality:** Search cars efficiently
- **Reviews & Messaging:** Users can send messages and give reviews for bookings
- **Wishlist / Subscription:** Subscribe to newsletters to receive special offers

### Admin Side
- **User Management:** Manage all users including admins, staff, and customers
- **Dashboard & Notifications:** Admin notifications for orders and updates
- **Order & Payment Management:** Track, update, and process orders and payments
- **CRUD Operations:** Add, update, delete cars, services, and spare parts
- **Recycling Bin:** Restore deleted items
- **Reports:** Generate reports with Excel, PDF, print, and charts
- **Security:** Change password, password reset for admin
- 
## Tech Stack
- **Backend:** Django 4.1
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Database:** MySQL
- **Payments:** Razorpay
- **Libraries:** openpyxl, reportlab, xhtml2pdf, Pillow, python-dotenv
